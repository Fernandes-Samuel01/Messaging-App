# 💬 Simple Web Messenger

A clean and modern web-based messenger built using **HTML, CSS, and JavaScript**.
This project demonstrates basic front-end development concepts like DOM manipulation, event handling, and responsive UI design.

---

## 🚀 Features

* 🎨 Visually appealing background
* 💬 Message display area with custom emoji bullets
* ⌨️ Input box with placeholder text
* 📨 Send button with emoji icon
* ⚡ Send messages using **Enter key**
* 🧼 Input field clears automatically after sending
* 📱 Responsive and modern UI (rounded corners, spacing, hover effects)

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout (Flexbox, gradients, shadows)
* **JavaScript (Vanilla JS)** – Functionality and interactivity

## 💡 How It Works

* User types a message in the input box
* Clicks the send button or presses **Enter**
* JavaScript:

  * Captures the input value
  * Creates a new `<li>` element
  * Appends it to the message list
  * Clears the input field
